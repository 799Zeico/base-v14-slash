# base-v14-slash

### Introduction
Ce code est un exemple de bot optimisé et organisé utilisant Discord.js v14, y compris les commandes Slash.

## Avant utilisation
- Install Node.js (>21)
- Install Visual Studio Code

## Installation
```sh
npm install
```
